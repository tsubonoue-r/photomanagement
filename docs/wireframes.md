## 2. 共通コンポーネント

### 2.1 ヘッダー

```
┌─────────────────────────────────────────────────────────────────────┐
│  [Logo] PhotoManagement    [検索...]           [通知] [?] [Avatar]  │
└─────────────────────────────────────────────────────────────────────┘

機能:
- ロゴ: ダッシュボードへ遷移
- グローバル検索: 写真・工事を横断検索
- 通知アイコン: 通知一覧を表示
- ヘルプアイコン: ヘルプ・ドキュメント表示
- アバター: プロフィールメニュー表示
```

### 2.2 サイドバー

```
┌────────────────────┐
│                    │
│  [工事名選択 ▼]    │
│                    │
├────────────────────┤
│                    │
│  □ ダッシュボード   │
│  □ 写真管理        │
│  □ 電子小黒板      │
│  □ 電子納品        │
│  □ 工種管理        │
│                    │
├────────────────────┤
│                    │
│  □ 設定            │
│  □ ヘルプ          │
│                    │
└────────────────────┘

機能:
- 工事選択: 現在の工事を切り替え
- ナビゲーション: 各機能へ遷移
- 折りたたみ対応: モバイルではハンバーガーメニュー
```

### 2.3 パンくずリスト

```
ダッシュボード > 工事一覧 > ○○道路改良工事 > 写真管理
```
